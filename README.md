# Ooru_Insights 🏙️
### Smart City Urban Analytics for Bengaluru

**Ooru_Insights** is an interactive Data Science dashboard built to visualize and analyze the urban pulse of Bengaluru. By leveraging Python and Streamlit, this project transforms raw city data (traffic, pollution, and infrastructure) into actionable insights.

## 🚀 Features
* **Live Traffic Heatmaps:** Visualizing congestion hotspots across the city.
* **Neighborhood KPI Tracker:** Real-time metrics for specific wards.
* **Urban Growth Trends:** Analyzing how the "Garden City" has evolved over time.
* **Interactive Filtering:** User-driven data exploration using Sidebar controls.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Libraries:** Streamlit, Pandas, Plotly/Matplotlib
* **Deployment:** Streamlit Cloud / Railway
* **Data Sources:** OpenCity Bengaluru / Kaggle
